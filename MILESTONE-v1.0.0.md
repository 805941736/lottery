# 双色球分析标注 v1.0.0 里程碑

日期：2026-06-23

## 版本定位

这是第一个主功能完整可用版本，后续修改默认以 `ssq-analysis.html` 当前实现为基础。

## 已固化功能

- 本地渲染最近 50 期双色球走势图。
- 最新一期号码随本地数据刷新显示。
- 蓝球趋势线、每 5 期分隔线。
- 红球横向、竖向、斜向 3 个及以上连号加深显示。
- 3 行预选号，任意工具状态下可点选。
- 画笔、直线、箭头、矩形、圆圈、文字标注。
- 双击标注后可移动、缩放，Delete 删除。
- Ctrl+Z 撤销，Ctrl+S 保存。
- 标注和预选号保存到浏览器 localStorage。
- 标注颜色固定为马卡龙蓝、绿、黄，线宽固定 3px。
- 可导出标注 PNG。
