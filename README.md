# 双色球分析标注

正式版入口：优先双击 `打开双色球分析.vbs`；如果系统限制 VBS，再双击 `打开双色球分析.cmd`。

## 目录

- `app/`：页面入口与标注工具界面。
- `data/`：本地开奖数据快照。
- `scripts/`：本地服务与数据刷新脚本。

## 使用

1. 双击根目录的 `打开双色球分析.vbs`。
2. 浏览器会在本地服务启动后自动打开页面；端口会自动选择，避免和旧服务冲突。
3. 点击“刷新图表”或顶部刷新按钮，可通过本地服务更新开奖数据。

标注记录会自动保存在浏览器本地；使用本地服务打开时，也会同步保存为 `data/ssq-analysis-records.json`，该文件不纳入版本管理。
